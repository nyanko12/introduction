## 2. グローバルUI要素（常時表示）

### 上部 — にゃんこ砲チャージバー（XPプログレス）

にゃんこ大戦争の「にゃんこ砲チャージゲージ」そのものとして表現する。

```
[ 🐱 BATTLE XP ========●------------ ] 042 / 100
```

- **動作**: スクロール量（`scrollY / documentHeight`）に比例し 0〜100 で増加
- **バー本体**: 黄色グラデーション風の単色塗り `#EF9F27` → チャージ中は点滅アニメ
- **アイコン**: バー左端ににゃんこの顔アイコン（SVGで自作 or 絵文字 🐱 で代替）
- **マイルストーン演出**:

| XP値 | 演出 |
|------|------|
| 25 | バーフラッシュ + `「にゃんこ砲、準備中…」` |
| 50 | バーフラッシュ + `「チャージ50%！」` |
| 75 | バーフラッシュ + `「もうすぐにゃんこ砲が撃てるにゃ！」` |
| 100 | バー全点灯 + `「🐱 にゃんこ砲、発射可能！」` + 画面端にエフェクト |

## 4. サウンドエフェクト設計（Web Audio API）

にゃんこ大戦争のSEを**Web Audio APIで合成**して再現する（著作権回避のため自前生成）。

| トリガー | SE内容 | 実装方法 |
|---------|--------|---------|
| ナビボタン クリック | 短い「ポコッ」音 | 矩形波 + 急速減衰 80ms |
| セクション進入 | 「ジャン」通知音 | サイン波 440Hz→880Hz グライド |
| XP 25/50/75達成 | 「チャリン」コイン音 | 高周波サイン波 複数重ね |
| XP 100達成 | にゃんこ砲発射SE風 | ホワイトノイズ + 低音バースト |
| ステージアンロック | 「ドン！」開放音 | 低周波サイン波 + ディケイ |
| GitHub ボタンホバー | 「ニャッ」短音 | ノコギリ波 200Hz 50ms |

```javascript
// SE生成サンプル（クリック音）
function playClickSE() {
  const ctx = new AudioContext();
  const osc = ctx.createOscillator();
  const gain = ctx.createGain();
  osc.connect(gain);
  gain.connect(ctx.destination);
  osc.type = 'square';
  osc.frequency.setValueAtTime(440, ctx.currentTime);
  gain.gain.setValueAtTime(0.3, ctx.currentTime);
  gain.gain.exponentialRampToValueAtTime(0.001, ctx.currentTime + 0.08);
  osc.start();
  osc.stop(ctx.currentTime + 0.08);
}
```

> **注意**: ユーザーの最初のクリック後にのみ AudioContext を生成する（ブラウザポリシー対応）。  
> ミュートボタンをナビ付近に設置し、デフォルトONにする。

## 5. アニメーション仕様

| 対象 | 仕様 |
|------|------|
| スクロール進入 | `IntersectionObserver` で各要素に `.visible` 付与 → fade-up + slide-in |
| XPバー | `scroll` イベント → `requestAnimationFrame` でバー幅・数値を更新 |
| ステータスバー | セクション①進入時に 0 → 実値トランジション（`0.8s ease-out`） |
| NEXT STAGESカード | shake → カラー化 → LOCKED→CLEAR 切り替え |
| ナビホバー | `transform: scale(1.08)` + Y軸移動 + `transition 100ms` |
| タイプライター | `setInterval` で1文字ずつ描画、末尾に `█` 点滅 |
| スキャンライン | CTAボタンの `::after` 疑似要素でホバー時に光が走る |

> パフォーマンス: CSS `transform` / `opacity` のみで合成レイヤーを維持。`will-change` は必要最小限に。
